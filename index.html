Notes for AI
  
  einnsyn        Public records, journal entries, government correspondence via eInnsyn API.
  stortinget     Parliamentary data: representatives, committees, votes, hearings.
  brreg          Norwegian company registry: org lookup, roles, board members via Brønnøysundregistrene API.
  entur          Public transport: journey planning, departures, stops via Entur APIs.
  geonorge       Place names, addresses, municipalities, coordinates via Kartverket APIs.
  kartkatalog    Spatial data catalogue: find datasets, WMS/WFS services on Geonorge.
  met-weather    Weather forecasts, nowcast, sunrise times via MET Norway API.
  nvdb           Road database: speed limits, road objects, tunnels, bridges via NVDB API.
  nve            Energy and water: hydropower, wind, reservoirs, flood/landslide warnings via NVE API.
  ssb            Statistics Norway: population, economy, demographics via StatBank and KLASS APIs.
  trafikkdata    Traffic volumes, ÅDT, hourly counts via Trafikkdata GraphQL API.
  vegbilder      Road imagery: street-level photos of Norwegian roads via Vegbilder WFS/S3.
